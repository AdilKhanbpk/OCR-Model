Quick Start Example

Use your API key to process documents programmatically:

curl -X POST https://api.ocrvision.com/v1/ocr \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf" \
  -F "languageHints=en,ur" \
  -F "searchablePdf=true"
